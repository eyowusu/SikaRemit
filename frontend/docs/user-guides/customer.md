# Customer Guide

Welcome to SikaRemit! This guide will help you understand how to use our platform for all your payment needs.

## Getting Started

### Registration and Login
1. Visit the SikaRemit website or mobile app
2. Click "Sign Up" and provide your details
3. Verify your email and phone number
4. Set up multi-factor authentication (MFA) for security
5. Log in using your credentials

### Account Verification
- Complete KYC (Know Your Customer) verification
- Upload required documents (ID, proof of address)
- Biometric verification may be required
- Verification typically takes 24-48 hours

## Core Features

### 1. Send Money
Send money to other SikaRemit users or external accounts.

**Via Web/App:**
1. Go to "Send Money" in your dashboard
2. Enter recipient's phone number or email
3. Enter amount and select payment method
4. Review and confirm transaction
5. Transaction completes instantly for SikaRemit users

**Via USSD:**
- Dial `*123*1*recipient*amount#`
- Example: `*123*10241234567*50#`

**Fees:** 1% + $1 for domestic transfers

### 2. Check Balance
View your account balance and transaction history.

**Via Web/App:**
- Dashboard shows current balance
- Navigate to "Account" > "Balance"

**Via USSD:**
- Dial `*123*2#`
- Response shows available balance

### 3. Pay Bills
Pay utility bills, internet, mobile top-ups, and more.

**Supported Billers:**
- Electricity (ECG, NEDCo)
- Water (GWCL)
- Internet (Vodafone, MTN)
- Mobile Airtime/Data
- TV subscriptions
- Taxes and levies

**Via Web/App:**
1. Go to "Pay Bills"
2. Select biller type
3. Enter bill reference/account number
4. Confirm payment

**Via USSD:**
- Dial `*123*3*biller*reference*amount#`

**Fees:** No fees for bill payments

### 4. Top Up Account
Add money to your SikaRemit wallet.

**Methods:**
- Mobile Money (MTN, Airtel, Vodafone)
- Bank Transfer
- Credit/Debit Card
- PayPal/Stripe

**Via Web/App:**
1. Go to "Top Up"
2. Select payment method
3. Enter amount
4. Complete payment

**Via USSD:**
- Dial `*123*4*amount#`

### 5. Cross-Border Transfers
Send money internationally with competitive exchange rates.

**Supported Countries:**
- Ghana, Nigeria, Kenya
- Other African countries
- Major global destinations

**Via Web/App:**
1. Go to "International Transfer"
2. Select destination country
3. Enter recipient details
4. Choose exchange rate
5. Confirm transfer

**Via USSD:**
- Dial `*123*5*country*recipient*amount#`

**Fees:** 2.5% + $5 for international transfers

### 6. Receive Invoice
Accept and pay invoices from merchants.

**Via Web/App:**
1. Check "Invoices" section
2. Review invoice details
3. Pay using preferred method

**Via USSD:**
- Dial `*123*6#` to view pending invoices

### 7. Transaction History
View all your past transactions.

**Via Web/App:**
- Go to "Transactions" or "History"
- Filter by date, type, amount
- Download statements

**Via USSD:**
- Dial `*123*7#`

### 8. Settings
Manage your account preferences.

**Options:**
- Update profile information
- Change password
- Manage payment methods
- Set up notifications
- Configure security settings

## Troubleshooting

### Common Issues

**Transaction Failed:**
- Check account balance
- Verify recipient details
- Ensure payment method is active
- Contact support if issue persists

**USSD Not Working:**
- Check network signal
- Ensure correct USSD code (*123*)
- Try again in a few minutes
- Contact your mobile provider

**Login Issues:**
- Reset password
- Check MFA settings
- Clear browser cache
- Contact support

### Disputing Payments
1. Find payment in History
2. Click 'Dispute'
3. Enter reason (select from options or write custom)
4. Upload supporting documents if needed
5. Submit

Disputes are processed within 48 hours.
